Linux Console Fonts Screenshots

Introduction

ArchLinux’s wiki page on console fonts was lacking an organized library of images available for previewing. So I made a Python script to produce a PNG image of all fonts contained in /usr/share/kbd/consolefonts. Then I semi-manually built this site using Ansible and Docsy.

Here is a tarball of the images : fontspng.tar.gz (1 359 545 bytes). It expands to a “fontspng” directory containing the same 205 PNG files as below.

Please send any comments, suggestions or questions to alexandre@deverteuil.net.
